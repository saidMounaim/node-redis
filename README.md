# node-redis

This project try to explain how you can use this three technologies togheter.

## Technology Stack:

- Node js
- Express Js
- MongoDB
- Redis

## Usage

"/.env" update the values/settings to your own

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run server

```
